# Syntecxhub_port_scanner
This project is a Python-based TCP Port Scanner that allows users to check for open, closed, or timed-out ports on a given host. It demonstrates the use of socket programming and threading/concurrency to perform network scanning efficiently.

The scanner can handle:

Scanning a single host for a range of ports

Logging the results of each port scan to a text file

Handling exceptions, timeouts, and errors gracefully

This tool is ideal for learning networking concepts, socket programming, and understanding how port scanning works in practice.
